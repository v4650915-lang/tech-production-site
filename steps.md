# 🚀 Поэтапный план разработки сайта РПК "Технология"

Этот файл содержит последовательные инструкции для реализации проекта. Выполняй шаги строго по порядку.

## Шаг 1: Инициализация и окружение
1. **Создание проекта:** Разверни Next.js 15 (App Router) с использованием TypeScript и Tailwind CSS v4.
2. **MCP Конфигурация:** - Используй `context7-mcp` для получения последних доков по Next.js 15 и Tailwind v4.
   - Используй `shadcn-ui-mcp-server` для установки базовых компонентов (button, input, card, dialog, sheet, toast).
3. **Настройка шрифтов:** Подключи `Montserrat` (для заголовков) и `Inter` (для текста) через `next/font`.

## Шаг 2: Глобальное состояние и лейаут
1. **Store:** Реализуй корзину (Cart) через React Context или Zustand. Корзина должна хранить параметры расчета (размеры, материал, доп. опции).
2. [cite_start]**Шапка и подвал:** Создай адаптивное меню и футер с контактными данными (Нижневартовск, ул. Мира, ЗП, стр. 1)[cite: 4].
3. **Анимации:** Настрой Framer Motion для плавного появления секций (fade-in-up).

## Шаг 3: Разработка "Витрины" (Showcase)
1. [cite_start]**Секции:** Создай карточки услуг для Широкоформатной печати, Плоттерной резки, Стендов и Сувениров[cite: 5, 13, 16, 41].
2. **Эффекты:** Реализуй "вау-эффект" при наведении на карточки (scale, светящиеся границы в стиле Tailwind v4).
3. **Фильтрация:** Добавь простую фильтрацию по категориям (Реклама / Резка / Сувениры).

## Шаг 4: Разработка интерактивных калькуляторов
1. [cite_start]**Калькулятор Баннеров:** Реализуй логику изменения цены в зависимости от объема (до и от 50м2)[cite: 6].
2. [cite_start]**Калькулятор Резки:** Внедрь расчет стоимости за погонный метр с учетом общей длины реза (порог 100 пог. м)[cite: 68].
3. **Интеграция с корзиной:** Кнопка "Добавить в корзину" должна передавать все выбранные параметры и итоговую сумму.

## Шаг 5: Корзина и Форма заказа
1. **Интерфейс корзины:** Список выбранных услуг с возможностью удаления.
2. **Валидация:** Используй React Hook Form + Zod для формы заказа (Имя, Телефон, Комментарий).
3. **Интеграция API:** Настрой отправку данных формы в Telegram Bot API.

## Шаг 6: Финализация и SEO
1. **SEO:** Настрой Metadata API для всех страниц.
2. **Оптимизация:** Проверь корректность `output: 'export'` для статического хостинга.
3. **Тестирование:** Проверь работу всех калькуляторов на соответствие прайс-листу.
